### USAGE
- base url:
  https://vidsrc-api-two.vercel.app

- endpoint:
  - `/{db_id}`

- parameters:
  - `s` - season (series only)
  - `e` - episodes (series only)

### EXAMPLE MOVIE
```
https://vidsrc-api-two.vercel.app/916224
```

### EXAMPLE SHOW
```
https://vidsrc-api-two.vercel.app/1429?s=1&e=1
```
